Detailed Description

template<typename IndependentVariableType, typename DependentVariableType, typename ScalarType = IndependentVariableType>
class tudat::interpolators::CubicSplineInterpolator< IndependentVariableType, DependentVariableType, ScalarType >

Cubic spline interpolator, implementation from (Press W.H., et al., 2002). Natural boundary conditions are imposed, meaning zero second derivatives (curvature) at end points. Continuity of first derivatives is imposed.

Template Parameters
IndependentVariableTypeType of independent variables.
DependentVariableTypeType of dependent variables.

void tudat::interpolators::CubicSplineInterpolator< IndependentVariableType, DependentVariableType, ScalarType >::calculateSecondDerivatives ( )
inlineprivate

This function calculates the second derivatives of the curve at the nodes, assuming the first derivatives to be continuous at the nodes and imposing natural spline conditions (zero curvature at endpoints). The methodology is described in (Press W.H., et al., 2002).
