Detailed Description

Class for calculating partial derivatives of a spherical harmonic gravitational acceleration, as calculated by the SphericalHarmonicsGravitationalAccelerationModel class.

Constructor, requires input on the acceleration model as of which partials are to be computed. If any partials of parameters of the rotation model of the body exerting acceleration are to be calculated, RotationMatrixPartial objects must be pre-constructed and passed here as a map, with one object for each parameter wrt which a partial is to be taken.

Parameters
acceleratedBodyName of body undergoing acceleration.
acceleratingBodyName of body exerting acceleration.
accelerationModelSpherical harmonic gravity acceleration model from which acceleration is calculated wrt which the object being constructed is to calculate partials.
rotationMatrixPartialsMap of RotationMatrixPartial, one for each paramater representing a property of the rotation of the body exerting the acceleration wrt which an acceleration partial will be calculated.
tidalLoveNumberPartialInterfacesList of objects to compute partials of tidal gravity field variations, one per corresponding variation object in acceleratedBody.
