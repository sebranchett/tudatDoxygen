Detailed Description

Tabulated atmospheres class, for example US1976. The default path from which the files are obtained is: /External/AtmosphereTables

◆ defaultExtrapolationValue_

std::vector< std::vector< std::pair< double, double > > > tudat::aerodynamics::TabulatedAtmosphere::defaultExtrapolationValue_
private

Default values to be used for extrapolation. The structure of the vector is as follows:

  • outer vector: one element for each dependent variable (density, pressure, temperature, ect.)
  • inner vector: one element for each independent variable (latitude, longitude and altitude)
  • pair: first element in case independent variable is below its lower definition limit, second element in case it is above the upper definition limit
