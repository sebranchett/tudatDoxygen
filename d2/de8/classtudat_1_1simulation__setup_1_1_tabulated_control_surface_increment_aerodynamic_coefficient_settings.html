Detailed Description

template<unsigned int NumberOfDimensions>
class tudat::simulation_setup::TabulatedControlSurfaceIncrementAerodynamicCoefficientSettings< NumberOfDimensions >

Object for setting aerodynamic coefficients from a user-defined N-dimensional table (with N>1). The user must provide the force (and moment) coefficients in boost multi_arrays, and define the physical meaning of each of the independent variables.
