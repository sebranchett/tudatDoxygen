Detailed Description

template<typename IndependentVariableType>
class tudat::statistics::ContinuousProbabilityDistribution< IndependentVariableType >

Base class for a continuous random variable, providing interfaces for evaluating probability and cumulative distribution functions (inverse cdf is only available for derived class InvertibleContinuousProbabilityDistribution). Specific functions are to be implemented by derived class. Derived class may be used for random number generation through ContinuousRandomVariableGenerator class.
