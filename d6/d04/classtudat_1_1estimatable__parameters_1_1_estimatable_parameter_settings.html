Detailed Description

Class for providing settings for parameters to estimate. This class is a functional base class for parameters that require no information in addition to their type. This class can be used for the easy setup of parameter objects (see createEstimatableParameters.h), but users may also chose to do so manually. (Derived) Class members are all public, for ease of access and modification.
