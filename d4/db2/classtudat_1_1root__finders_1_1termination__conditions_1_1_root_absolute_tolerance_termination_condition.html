Detailed Description

template<typename ScalarType = double>
class tudat::root_finders::termination_conditions::RootAbsoluteToleranceTerminationCondition< ScalarType >

This class implements the TerminationCondition base class to check if the absolute tolerance for the root value computed by the root-finder that this class is used in conjunction with has been reached. Additionally, this termination condition also checks if the maximum number of iterations has been exceeded. This class effectively serves as a wrapper for the checkRootAbsoluteTolerance()- and checkMaximumIterationsExceeded()-functions.

See also
checkRootAbsoluteTolerance(), checkMaximumIterationsExceeded().
