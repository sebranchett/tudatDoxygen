The methods outlined by e.g. Dirkx et al. (2016); Planetary and Space Science 134:82-95. This function uses a nominal dynamical model to simulate ideal observations of a set of N bodies' 3-dimensional positions. These observations are then used as input to an orbit determination routing in which a set of parameters have their values adjusted w.r.t. the nominal case. In the orbit determination, only the initial states of the N bodies are estimated.
