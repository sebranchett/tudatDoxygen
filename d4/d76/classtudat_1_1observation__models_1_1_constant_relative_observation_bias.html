Detailed Description

template<int ObservationSize = 1>
class tudat::observation_models::ConstantRelativeObservationBias< ObservationSize >

Class for a constant relative observation bias of a given size. For unbiases observation h and bias A, the biased observation is computed as h .* A, where .* is the component-wise multiplication.
