template<typename IndependentVariableType = double, typename DependentVariableType = double>
class tudat::filters::KalmanFilterBase< IndependentVariableType, DependentVariableType >

Base class for the set up and use of Kalman filters.

Template Parameters
IndependentVariableTypeType of independent variable. Default is double.
DependentVariableTypeType of dependent variable. Default is double.

integratorSettingsSettings for the numerical integrator to be used to propagate state.
