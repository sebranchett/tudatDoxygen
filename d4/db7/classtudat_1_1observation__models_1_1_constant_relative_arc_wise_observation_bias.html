Detailed Description

template<int ObservationSize = 1>
class tudat::observation_models::ConstantRelativeArcWiseObservationBias< ObservationSize >

Class for an arc-wise constant relative observation bias of a given size. For unbiases observation h and bias A, the biased observation is computed as h .* A, where .* is the component-wise multiplication.The bias A is provided per arc, with the arc start times provided to the class constructor.
