Detailed Description

Class that uses random samples to generate a multivariate probability distribution using Kernel Density distribution. The bandwidth of the kernels may be supplied by the user, or an optimal distrubution may be computed by this class At present, the user has the choice of a Gaussian or Epanechnikov distribution for the kernels.

Parameters
samplesData samples from which the empirical distribution is to be created.
bandWidthFactorFactor by which to multiply the computed bandwidth values of the kernels.
kernel_typeType of distribution to be used for the kernels in the distribution.
manualStandardDeviationStandard deviations with which to scale the entries of each sample. By default this vector is empty and not used. If it is provided, it is used to scale the entry i of each sample by manualStandardDeviation( i ) / sampleStandardDeviation( i ), where sampleStandardDeviation is the standard deviation computed from the samples/
manualBandwidthVector of bandwidths for each dimension that is to be used in the kernels. By default this vector is empty and not used. Optimal bandwidths (scaled by bandWidthFactor) are used in this default case.
