Detailed Description

A class that calculates the required impulses for a swingby leg for an MGA-1DSM position formulation trajectory model. This model specifies the trajectory by defining the location and the moment of application of the DSM. This results in two sublegs, which are then solved using two Lambert targeters. A gravity assist module is then used for the first of the two legs. Typically the user will be primarily interested in the total delta V and the arrival velocity of the spacecraft before the arrival planet.
