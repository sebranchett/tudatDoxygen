Detailed Description

template<typename DirectAccelerationModelType, typename std::enable_if< is_direct_gravity_acceleration< DirectAccelerationModelType >::value, int >::type = 0>
class tudat::gravitation::ThirdBodyAcceleration< DirectAccelerationModelType, type >

Class for calculating third-body (gravitational accelerations), i.e. the gravitational acceleration on a body A, expressed in a frame fixed on a body B, due to a gravitating body C. The acceleration is calculated by subtracting the acceleration of the central body (B) due to body C from the direct acceleration of body C on body A.

Template Parameters
Thegravitational acceleration class for which the third body acceleration is calculated (CentralGravitationalAccelerationModel, SphericalHarmonicsGravitationalAccelerationModel, ...)
