Detailed Description

A TextParser provides a set of functions to streamline parsing to simplify the parsing process. The inheriting parsers can chose there prefered way of processing by calling TextParser(bool).

NOTE: This TextParser works with the FieldValue/FieldType architecture. For simpler file reading, use, for instance, matrixTextFileReader.
