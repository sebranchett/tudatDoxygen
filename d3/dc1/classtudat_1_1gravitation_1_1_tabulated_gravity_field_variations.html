Detailed Description

Class for variations in spherical harmonic coefficients, directly interpolated from tables. Maps of variatons of cosine and sine coefficients as a function of time are provided, as well as the type of interpolator that is to be used, from which interpolated variations are then returned at any time. The class may also be used for defining the complete gravity field (i.e. not only the variations) as a function of time when the constant coefficient contribution (i.e. not from a GravityFieldVariations) is zero for all degrees and orders.

Class constructor, takes the maps of variations, as well as the minimum degree and order of the corrections, the corrections are applied as a rectangular block in degree and order, with the the size of the blocks determined by the contents of the maps. The matrix size of all map entries must be equal for both sine and cosine values.
