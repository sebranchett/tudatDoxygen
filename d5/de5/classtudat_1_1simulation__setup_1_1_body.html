Detailed Description

Body class representing the properties of a celestial body (natural or artificial). By storing all properties of bodies (ephemeris, rotation, gravity, etc.) in a set of body objects, the simulation environment can be defined in a clear and modular way. To create body objects, the createBodies.h function provides a range of functionality. The createAccelerationModels.h file provides functions to use body objects to create acceleration objects.
