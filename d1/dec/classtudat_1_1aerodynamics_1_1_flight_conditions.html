Detailed Description

Class for calculating aerodynamic flight characteristics of a vehicle during numerical integration, in the absence of an atmosphere.

Class is used to ensure that dependent variables such as altitude, etc. are only calculated once during each numerical integration step. The get functions of this class are linked to the various models in the code that subsequently require these values. In the case of atmospheric flight, the AtmosphericFlightConditions derived class should be used.
