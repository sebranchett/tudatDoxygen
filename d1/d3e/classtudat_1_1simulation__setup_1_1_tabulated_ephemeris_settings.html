Detailed Description

EphemerisSettings derived class for defining settings of an ephemeris created from tabulated data.

Currently the use of an 6th order Lagrange interpolator is hardcoded, which is created from the data that is provided. Note that at the edges of the interpolation interval, a Cubic spline interpolator is used to suppres the influence of Runge's phenomenon.
