Constructor with no specific gas constant nor ratio of specific heats, and with single boundary handling parameters.

These two values will be given the default Earth value, or are specified inside the atmosphere table file (and thus, inside the dependent variables vector). Only one boundary handling parameter is specified, which is then repeated for dimension.

Parameters
atmosphereTableFileMap of files containing information on the atmosphere. The order of both independent and dependent parameters needs to be specified in the independentVariablesNames and dependentVariablesNames vectors, respectively. Note that specific gas constant and specific heat ratio will be given the default constant values for Earth, unless they are included in the file map.
independentVariablesNamesList of independent parameters describing the atmosphere.
dependentVariablesNamesList of dependent parameters output by the atmosphere.
boundaryHandlingMethod for interpolation behavior when independent variable is out of range.
defaultExtrapolationValueDefault value to be used for extrapolation, in case of use_default_value or use_default_value_with_warning as methods for boundaryHandling.
